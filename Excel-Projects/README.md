# Financial Performance Dashboard (Excel)

## Objective
Create an interactive dashboard to visualize business performance using Excel.

## Skills Demonstrated
- Pivot tables
- Excel formulas (SUMIFS, AVERAGEIFS, XLOOKUP)
- Charts & dashboards
- Data cleaning

## Dataset
- File: `Datasets/retail_sales.csv`

## Insights (to be included in dashboard)
1. Monthly revenue trends
2. Revenue by product category
3. Top customers by revenue
