# Retail Sales Analysis (MySQL)

## Objective
Analyze retail sales data to extract business insights using MySQL.

## Skills Demonstrated
- Database creation
- Table design
- Aggregate functions (SUM, GROUP BY)
- Ordering & ranking
- Business analytics queries

## Key Queries
1. Total revenue
2. Revenue by month
3. Top products by revenue
4. Customer lifetime value
5. Revenue by category

## Dataset
- File: `Datasets/retail_sales.csv`
